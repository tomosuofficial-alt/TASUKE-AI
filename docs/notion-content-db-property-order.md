# コンテンツDB — プロパティの並び順（編集しやすい順）

DB ID: `942d70a4-e645-464e-a1ab-5176bce10939`

## 注意

**Notion 公開 API では「列の表示順」を変更できません。**  
並び替えは **Notion アプリ上で手動**です（各ブロック内の **# の順**に列をドラッグ）。

### Notion 画面でも見やすくしたい場合

表の列の「間」に区切りは引けないので、**同じ DB で「見る画面」を2つ作る**（ビュー複製）のが簡単です。  
**やさしい手順だけ** → **`notion-content-db-notion-ui-layout.md`**（上の「まずここだけ」から読める）

### 手動のやり方（フルページ DB の場合）

1. データベースを開く（表ビュー）
2. 列見出しを **左右にドラッグ**して順番を変える  
   または: 列の **+** の左、「…」メニュー → **プロパティ** から表示順を調整

## 補足

- 実際の DB に **上記にないプロパティ**（ロールアップ・数式など）がある場合は、**F の後**か、用途に合わせて A〜E の近くに置くとよいです。
- **狭い画面**では **A〜E だけ表示**し、**F は折りたたみ／非表示**がおすすめです。→ **ビュー分割**なら `notion-content-db-notion-ui-layout.md` の「編集 A〜E」「インサイト F」がそのまま使えます。
