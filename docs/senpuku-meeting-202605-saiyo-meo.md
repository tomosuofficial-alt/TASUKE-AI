# Part B：MEO（Googleマップ最適化）

## B-1. なぜ今MEOか（3分）

- **来店動線**：「居酒屋 高崎」「高崎 個室 居酒屋」「高崎 接待」等のGoogleマップ検索は、千福の客層（30〜50代、宴会・接待・特別な日利用）と相性が良い
- **千福の強み**：客単価1万円帯の質ある居酒屋 → **写真・クチコミの良し悪しが意思決定に直結**する業態
- **今ある資産を活かす**：3月公開の台湾インフルエンサー動画の認知波及を、Instagramだけでなく **GBP側の写真・最新情報にも**反映させたい
- **今やる理由**：MEOは「整えてから打つ」と効く。歓送迎会期（6月）と夏需要（7-8月）の前に基礎を作る

事前Web調査により、以下4点は**実機GBPを見る前に「ほぼ確実に改善が必要」**:

### 進行メモ（B-2）

- **担当列は AI幹部チーム**：ミセル（Content）、ヒビキ（CMO）、マワリ（COO）、サグル（CDO）、ツクル（CTO）、ツカサ（CEO）が、それぞれの専門領域でGBP項目を担う。**会議の場では大内が代理確認**、項目ごとの実行依頼は会議後にツカサからサブエージェントへ委譲する
